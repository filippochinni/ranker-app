# Ranker App

Ranker-App is an utility app for building Rankings and other related utility functionalities.<br>
It can be used either as a command line application or navigated through a menu interface.


## Usage

### Menu Interface Usage

| Menu | Functionality | Description |
| --- | --- | --- |
| Ranking Musica | Build Ranking Canzoni | ... |
| Ranking Musica | Build Ranking Artisti | ... |
| Ranking Musica | Build Ranking Canzoni per Artista | ... |
| Ranking Anime | Build Ranking Anime | ... |
| Ranking Anime | Build Lista Nomi Anime | ... |
| Ranking Film | Build Ranking Film | ... |
| Ranker Utilities | Index Ranking | ... |
| Ranker Utilities | Playlists to CSV | ... |
| Music Utilities | Folders to Playlists | ... |
| Music Utilities | Playlists Fixer | ... |

### Command Line Usage
```
ranker-app [-h] [-cl]
```
#### Options

`-h`, `--help` &emsp;&emsp;&emsp;&emsp; Show this help message

`-cl` &emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;&emsp; Use the Command Line interface instead of the Menu Interface
